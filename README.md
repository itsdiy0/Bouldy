## What is Bouldy?

Bouldy enables users to upload documents (PDF, DOCX, TXT), create custom AI chatbots trained on those documents, and deploy chatbots via embeddable widgets or shareable links using their own LLM API keys (OpenAI, Anthropic, etc.)

Built with RAG (Retrieval-Augmented Generation) using LlamaIndex.
